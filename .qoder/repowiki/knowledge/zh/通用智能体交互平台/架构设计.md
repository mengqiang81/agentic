- **协议桥接**：前端 Copilot Runtime (`frontend/app/api/copilotkit/route.ts`) 作为 BFF 层，将请求透传至后端 `/awp/v1/runs` 端点，实现 AG-UI 标准协议的端到端对齐。
- **多租户路由**：利用 `X-Tenant-Id` 和 `X-User-Id` HTTP 头在前后端之间传递上下文，后端 `AgUiController` 据此构建 `RuntimeContext` 并隔离 Redis 会话存储。
- **HITL 状态同步**：后端通过 SSE 推送 `interrupt` 事件触发前端审批组件，前端通过 `resume` 字段回传决策，后端 `AgentEventMapper` 负责将内部事件转换为标准 AG-UI 格式。
