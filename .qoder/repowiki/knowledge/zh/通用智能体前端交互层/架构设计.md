- **框架选型**：采用 Next.js (App Router) 作为基础架构，集成 CopilotKit (`@copilotkit/react-ui`, `@copilotkit/runtime`) 实现对话 UI 与运行时逻辑。
- **代理桥接**：在 `app/api/copilotkit/route.ts` 中定义 Next.js API Route，利用 `HttpAgent` 将前端请求透传至后端 AG-UI 端点，并注入租户与用户身份头信息。
- **性能优化**：通过自定义 `FastAgentRunner` (`lib/fast-agent-runner.ts`) 替换默认 Runner，跳过冗余的 `compactEvents` 处理，确保 SSE 流及时关闭以解锁前端 UI。
- **HITL 交互**：利用 `InterruptApproval` 组件配合 `useLangGraphInterrupt` Hook，监听后端中断事件并渲染确认/拒绝按钮，实现标准 interrupt/resume 机制。
- **状态管理**：当前仅支持 Session 级工作区，线程持久化接口 (`threads/route.ts`) 暂返回空数据，依赖后端 RedisDistributedStore 维护全量状态。
