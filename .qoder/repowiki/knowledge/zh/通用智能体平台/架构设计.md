- **前后端通信**：前端通过 `@copilotkit/runtime` 在 Next.js API Route (`/api/copilotkit`) 中建立代理，将租户上下文（`x-tenant-id`）透传至后端。
- **协议适配**：后端基于 Spring WebFlux 暴露 SSE 流式端点，并通过 `AgentEventMapper` 将 AgentScope 事件映射为标准 AG-UI 协议格式。
- **状态隔离**：利用 Redis 实现多租户 Session 持久化，配合文件系统按 `tenant:user:agent:session` 层级组织工作区数据。
- **启动编排**：后端由 Spring Boot 统一管理 AgentScope Harness、DashScope Starter 及 OTEL 组件的初始化；前端通过环境变量动态配置后端地址。
